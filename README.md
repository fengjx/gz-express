# 广州地铁快车时刻表小程序，前端 + 后端

使用微信云托管开发部署示例项目

技术栈：[tdesign](https://tdesign.tencent.com/miniprogram/overview) + golang。

可以作为小程序、go web、云托管入门学习案例。

## 工程结构

```
- gzeweapp: 小程序前端工程
- gzesvr：小程序服务端工程
```

服务端工程仅用于测试演示，实际开发你需要自己实现动态获取数据的逻辑，例如从数据库读取数据。
